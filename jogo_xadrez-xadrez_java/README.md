# Jogo de Xadrez versão em Java

## Como baixar o projeto
 
### Git

#### git clone https://gitlab.com/ricardo.jmm/jogo_xadrez.git

#### Realizando o download

##### Clica no link https://gitlab.com/ricardo.jmm/jogo_xadrez/-/archive/xadrez_java/jogo_xadrez-xadrez_java.zip

##### Descompacta o arquivo jogo_xadrez-xadrez_java.zip

### Importando o projeto no eclipse

#### Dentro do eclipse abra a pasta onde está a pasta do jogo de xadrez

#### Clicar File -> Import -> Existing Project Into Workspace

#### Clicar em Finish
